
          	 ФИТНЕС ТРАКЕР  


ОПИСАНИЕ

Конзолно приложение за проследяване на тренировки,
написано на Python. Позволява на потребителя да създава
тренировъчни сесии, да добавя упражнения и да анализира
резултатите си.


СТРУКТУРА НА ПРОЕКТА

fitness_tracker/
models.py         - Клас WorkoutSession с всички методи
main.py           - Стартиращ файл с меню
requirements.txt  - Използвани библиотеки
readme.txt        - Този файл


КАК СЕ СТАРТИРА

Изисква Python 3.10 или по-нова версия.

  python main.py


ИЗПОЛЗВАНИ БИБЛИОТЕКИ

- datetime (вградена) — работа с дати на тренировките


КЛАС: WorkoutSession (models.py)

Конструктор:
  WorkoutSession(name, workout_date=None)

Методи:
  add_exercise(name, sets, reps, calories)
      Добавя упражнение към сесията.

  set_duration(minutes)
      Задава продължителността на тренировката.

  get_summary() -> dict
      Връща пълно обобщение на сесията.

  get_calories_per_minute() -> float
      Изчислява интензивността (ккал/мин).

  find_hardest_exercise() -> dict | None
      Намира упражнението с най-много калории.

  sort_exercises_by_calories(descending=True) -> list
      Сортира упражненията по изгорени калории.

  filter_by_min_sets(min_sets) -> list
      Филтрира упражненията по минимален брой серии.


ФУНКЦИОНАЛНОСТИ В МЕНЮТО (main.py)

  1. Нова тренировъчна сесия
  2. Добави упражнение
  3. Задай продължителност
  4. Виж обобщение + интензивност
  5. Най-тежко упражнение
  6. Сортирай упражненията по калории
  7. Филтрирай по минимален брой серии
  0. Изход
